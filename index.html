Bidang yang dibina

Programming

Bidang ini menguji keterampilan peserta untuk menyelesaikan masalah algoritmik dalam tim menggunakan bahasa seperti C, C++, Python, atau Java. Kompetisi berbasis ICPC ini mencakup struktur data, algoritma, matematika, dan teori graf. Strategi utama untuk menang adalah latihan soal, penguasaan algoritma, kerja sama tim, dan manajemen waktu. Kompetisi ini bermanfaat untuk meningkatkan problem-solving, memperkuat pemahaman pemrograman, serta membuka peluang karier di industri teknologi.

Cyber Security

Dalam bidang Cyber Security, peserta akan diuji dalam melakukan penetration testing, cryptography, digital forensics, dan network security melalui kompetisi CTF (Capture The Flag). Tim diminta untuk menyelesaikan tantangan seperti web security, reverse engineering, dan exploit development. Strategi utama adalah latihan di platform CTF, menguasai keamanan jaringan, serta menggunakan tools seperti Wireshark dan Burp Suite. Kompetisi ini bermanfaat untuk meningkatkan keterampilan cybersecurity dan membuka peluang karier sebagai Pentester atau Security Engineer.

UX Design

Bidang UX Design di GEMASTIK menantang peserta untuk merancang pengalaman pengguna (User Experience) yang intuitif, fungsional, dan menarik. Peserta diuji untuk menganalisis kebutuhan pengguna, membuat wireframe, prototipe, dan menguji desain berbasis prinsip usability, accessibility, dan user-centered design.

Software Development

Bidang Software Development di GEMASTIK menantang peserta untuk membangun perangkat lunak inovatif yang efisien dan user-friendly. Peserta dituntut untuk mengembangkan solusi digital dengan menerapkan pengkodean, arsitektur perangkat lunak, dan deployment menggunakan teknologi modern seperti React, Flutter, atau Node.js

Game Development

Bidang Games Development di GEMASTIK menguji kemampuan peserta dalam merancang dan mengembangkan game yang menarik, interaktif, dan berkualitas. Dalam melakukan tugasnya, peserta diminta untuk membangun game menggunakan engine seperti Unity atau Unreal Engine, dengan fokus pada gameplay, desain grafis, dan pengalaman pengguna.

Tentang Gemastik Informatics Training 2025

Gemastik Informatics Training adalah sebuah program pelatihan yang diselenggarakan oleh divisi Keilmuan dan Teknologi HMIF Unsoed dengan tujuan untuk mempersiapkan para delegasi agar siap berkompetisi di ajang Gemastik 2025. Program ini mencakup berbagai bidang yang dirancang untuk meningkatkan keterampilan teknis dan pengetahuan peserta dalam bidang teknologi informasi.

Melalui pelatihan intensif ini, para peserta tidak hanya akan mendapatkan pembekalan berupa teori, tetapi juga pengalaman praktis yang sangat berguna dalam menghadapi kompetisi di tingkat nasional. Selain itu, pelatihan ini juga bertujuan untuk menciptakan kolaborasi dan semangat tim yang tinggi, yang menjadi kunci utama dalam mencapai keberhasilan di Gemastik.
